💡 使用範例

CSS變數使用

/* 使用主色調 */
.primary-button {
  background: var(--primary-8eccfc);
  color: var(--black-000000);
}

/* 使用透明度 */
.overlay {
  background: var(--black-38);
}

/* 使用系統色彩 */
.error-message {
  background: var(--error-cf6679);
  color: var(--white-87);
}
