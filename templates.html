使用說明

步驟 1：下載範本

選擇適合您專案需求的範本，點擊「下載範本」按鈕。

步驟 2：解壓縮檔案

將下載的 ZIP 檔案解壓縮到您的專案目錄。

步驟 3：引入慈濟設計系統

確保在 HTML 檔案中正確引入慈濟設計系統的 CSS 檔案：

<link rel="stylesheet" href="path/to/tzu-chi-design-system.css">

步驟 4：自訂內容

根據您的需求修改範本中的內容、顏色和版面配置。所有範本都支援慈濟設計系統的 CSS 變數，方便您進行主題自訂。

/* 自訂主題顏色 */
:root {
    --tzuchi-primary: #1B2D58;
    --tzuchi-accent: #5DC5C4;
    /* 更多自訂變數... */
}

範本結構

每個範本包含以下檔案：

  • • index.html - 主頁面
  • • styles.css - 範本特定樣式
  • • scripts.js - 互動功能（如有需要）
  • • README.md - 範本說明文件
  • • assets/ - 圖片和其他資源
