# 更新日誌

## [1.0.1] - 2025-06-18

### 新增
- 完整的 CIS (慈濟大入口網) Dark Mode 設計系統
- 基於 CSS 變數的主題架構
- 響應式 12 欄網格系統
- 全新元件庫：
  - 按鈕系統（多種變體和狀態）
  - 卡片系統（支援多種佈局）
  - 徽章系統（含 SDGS 色彩）
  - 表單元件（輸入框、單選框、複選框、開關）
  - 搜尋欄（含下拉建議）
  - 下拉選單（支援多方向展開）
- 完整的文字排版系統
- CSS Reset 和基礎樣式

### 改進
- 模組化的 CSS 架構
- 更好的可維護性
- 預留 Light Mode 擴充空間
