# 更新日誌

## [1.0.2] - 2025-06-18

### 新增
- 完整的圖標系統組件 (Icons System)
  - 支援 6 種尺寸：18px, 20px, 24px, 32px, 40px, 48px
  - 圖標顏色系統：品牌色、功能色
  - 圖標按鈕組件
  - 社群媒體圖標樣式
  - 特殊圖標：點狀、更多選項、漢堡選單、關閉、箭頭
  - 圖標動畫：旋轉、脈動效果
  - 響應式調整

### 改進
- 更新組件展示頁面，加入圖標系統完整範例
- 優化首頁展示，加入圖標系統展示區

## [1.0.1] - 2025-06-18

### 新增
- 完整的 CIS (慈濟大入口網) Dark Mode 設計系統
- 基於 CSS 變數的主題架構
- 響應式 12 欄網格系統
- 全新元件庫：
  - 按鈕系統（多種變體和狀態）
  - 卡片系統（支援多種佈局）
  - 徽章系統（含 SDGS 色彩）
  - 表單元件（輸入框、單選框、複選框、開關）
  - 搜尋欄（含下拉建議）
  - 下拉選單（支援多方向展開）
- 完整的文字排版系統
- CSS Reset 和基礎樣式

### 改進
- 模組化的 CSS 架構
- 更好的可維護性
- 預留 Light Mode 擴充空間

## [1.0.0] - 2025-06-17

### 初始版本
- 基礎設計系統框架
- 色彩系統
- 基本元件
