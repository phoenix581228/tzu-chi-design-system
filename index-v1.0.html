使用指南 Usage Guide

1. 引入 CSS 檔案

<link rel="stylesheet" href="tzu-chi-design-system.css">

2. 使用 CSS 變數

/* 在您的 CSS 中使用變數 */
.my-element {
    background-color: var(--tzuchi-bg-surface);
    color: var(--tzuchi-text-primary);
    border: 1px solid var(--tzuchi-divider);
}

3. 應用類別名稱

<button class="tzuchi-btn tzuchi-btn-primary">
    點擊我
</button>

<div class="tzuchi-card">
    <div class="tzuchi-card-body">
        卡片內容
    </div>
</div>
