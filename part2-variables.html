Part 2 - Variables

CIS 設計系統 - 主題變數定義
所有主題共用的基礎變數，包含字型、間距、圓角、陰影等

🔤 字型系統

多語言支援的字型堆疊定義

--cis-font-family-base
-apple-system, BlinkMacSystemFont, "Segoe UI", "Microsoft JhengHei", "微軟正黑體", "PingFang TC", "Helvetica Neue", Arial, sans-serif
主要字型預覽 Main Font Preview
--cis-font-family-mono
"SF Mono", "Monaco", "Inconsolata", "Fira Code", monospace
等寬字型預覽 Mono Font Preview

💡 使用範例

字型系統使用

/* 使用字型變數 */
.title {
  font-family: var(--cis-font-family-base);
  font-size: var(--cis-font-size-2xl);
}

.code {
  font-family: var(--cis-font-family-mono);
  font-size: var(--cis-font-size-sm);
}

間距與圓角使用

/* 使用間距變數 */
.card {
  padding: var(--cis-spacing-4);
  margin: var(--cis-spacing-2);
  border-radius: var(--cis-radius-lg);
  box-shadow: var(--cis-elevation-2);
}

/* 使用動畫變數 */
.button {
  transition: all var(--cis-duration-200) var(--cis-ease-out);
}
