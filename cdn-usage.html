🚀 快速開始

選擇以下任一 CDN 服務，將連結添加到您的 HTML 頁面中：

jsDelivr CDN 推薦

全球 CDN 網絡，自動選擇最快的節點

<!-- 最新版本 -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/tzu-chi-design-system@latest/tzu-chi-design-system.min.css">

<!-- 指定版本 -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/tzu-chi-design-system@1.0.0/tzu-chi-design-system.min.css">

<!-- 非壓縮版本（開發用） -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/tzu-chi-design-system@latest/tzu-chi-design-system.css">

unpkg CDN

快速、可靠的 npm CDN 服務

<!-- 最新版本 -->
<link rel="stylesheet" href="https://unpkg.com/tzu-chi-design-system@latest/tzu-chi-design-system.min.css">

<!-- 指定版本 -->
<link rel="stylesheet" href="https://unpkg.com/tzu-chi-design-system@1.0.0/tzu-chi-design-system.min.css">

<!-- 瀏覽套件內容 -->
<!-- 訪問: https://unpkg.com/browse/tzu-chi-design-system@latest/ -->

GitHub Pages

直接從 GitHub 託管的版本

<!-- 主分支最新版本 -->
<link rel="stylesheet" href="https://tzu-chi.github.io/design-system/tzu-chi-design-system.min.css">

<!-- 特定版本標籤 -->
<link rel="stylesheet" href="https://tzu-chi.github.io/design-system/v1.0.0/tzu-chi-design-system.min.css">

Netlify CDN

高性能邊緣網絡，支援即時部署

<!-- 生產環境 -->
<link rel="stylesheet" href="https://tzu-chi-design-system.netlify.app/tzu-chi-design-system.min.css">

<!-- 預覽部署（用於測試） -->
<link rel="stylesheet" href="https://deploy-preview-{number}--tzu-chi-design-system.netlify.app/tzu-chi-design-system.min.css">

🛡️ 安全性建議

使用 SRI (Subresource Integrity)

<!-- 使用 SRI 確保檔案完整性 -->
<link rel="stylesheet" 
      href="https://cdn.jsdelivr.net/npm/tzu-chi-design-system@1.0.0/tzu-chi-design-system.min.css"
      integrity="sha384-oqVuAfXRKap7fdgcCY5uykM6+R9GqQ8K/uxy9rx7HNQlGYl1kPzQho1wx4JwY8wC"
      crossorigin="anonymous">

<!-- 生成 SRI hash -->
<!-- 使用線上工具：https://www.srihash.org/ -->
<!-- 或使用命令行：openssl dgst -sha384 -binary file.css | openssl base64 -A -->
⚠️ 注意：使用 @latest 標籤時無法使用 SRI，因為檔案內容可能會變更。生產環境建議使用固定版本號。

📦 NPM 安裝

除了 CDN，您也可以通過 npm 安裝：

# 使用 npm
npm install tzu-chi-design-system

# 使用 yarn
yarn add tzu-chi-design-system

# 使用 pnpm
pnpm add tzu-chi-design-system

在項目中引入

// 在您的主 CSS 或 JavaScript 檔案中
import 'tzu-chi-design-system/tzu-chi-design-system.min.css';

// 或在 CSS 中
@import 'tzu-chi-design-system/tzu-chi-design-system.min.css';

🔧 故障排除

💡 提示：如果 CDN 載入失敗，可以使用以下回退方案： 
<!-- CDN 回退方案 -->
<link rel="stylesheet" 
      href="https://cdn.jsdelivr.net/npm/tzu-chi-design-system@latest/tzu-chi-design-system.min.css"
      onerror="this.onerror=null;this.href='https://unpkg.com/tzu-chi-design-system@latest/tzu-chi-design-system.min.css';">

<!-- 或使用 JavaScript 檢測 -->
<script>
// 檢查 CSS 是否載入成功
function checkCSS() {
    const testEl = document.createElement('div');
    testEl.className = 'tzu-chi-test';
    document.body.appendChild(testEl);
    const styles = window.getComputedStyle(testEl);
    
    if (!styles.getPropertyValue('--tzu-chi-primary')) {
        // CSS 未載入，使用備用方案
        const link = document.createElement('link');
        link.rel = 'stylesheet';
        link.href = '/local/tzu-chi-design-system.min.css';
        document.head.appendChild(link);
    }
    
    document.body.removeChild(testEl);
}

// DOM 載入完成後檢查
document.addEventListener('DOMContentLoaded', checkCSS);
</script>
