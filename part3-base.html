Material Design 基礎樣式
基礎重置和全域樣式定義，建立一致的設計基礎

🔄 CSS 重置

統一瀏覽器預設樣式，建立一致的基礎

📦

Box Model 統一

所有元素使用 border-box，確保尺寸計算一致

box-sizing: border-box;
🎯

邊距重置

清除瀏覽器預設的 margin 和 padding

margin: 0; padding: 0;
📝

字體繼承

表單元素繼承父元素字體設定

font-family: inherit;
🖼️

響應式圖片

圖片自動適應容器大小

max-width: 100%;

CSS 重置代碼

/* 基礎重置 */
*,
*::before,
*::after {
  box-sizing: border-box;
  margin: 0;
  padding: 0;
}

/* HTML 基礎設定 */
html {
  font-size: 16px;
  line-height: 1.5;
  -webkit-text-size-adjust: 100%;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
  scroll-behavior: smooth;
}

🎨 Body 全域樣式

設定頁面的基礎外觀和主題切換

全域樣式展示

這是 body 元素的基礎樣式，包含：

  • 字體系統：使用 CI 字型變數
  • 背景色彩：響應主題變數
  • 平滑轉場：支援主題切換動畫
  • 色彩繼承：文字顏色自動適應

Body 樣式代碼

body {
  margin: 0;
  padding: 0;
  font-family: var(--font-primary);
  background-color: var(--background);
  color: var(--on-background);
  transition: background-color var(--cis-duration-300) var(--cis-ease-out),
              color var(--cis-duration-300) var(--cis-ease-out);
}

🎯 焦點管理與無障礙性

現代化的焦點指示器，改善鍵盤導航體驗

使用 Tab 鍵聚焦此按鈕

鍵盤聚焦時顯示焦點環

可聚焦連結

連結元素的焦點狀態

焦點樣式代碼

/* 移除預設焦點 */
:focus {
  outline: none;
}

/* 鍵盤焦點指示器 */
:focus-visible {
  outline: 2px solid var(--primary-500);
  outline-offset: 2px;
}

✨ 選取文字樣式

自訂文字選取的背景和顏色

請選取這段文字來查看自訂的選取樣式效果。選取時會顯示品牌主色作為背景，確保文字的可讀性和品牌一致性。

📜 自訂捲軸樣式

現代化的捲軸設計，與整體主題一致

可捲動內容示範

這是一個可捲動的容器，展示自訂的捲軸樣式。

捲軸使用主題色彩，提供一致的視覺體驗。

在不同主題下，捲軸顏色會自動適應。

請嘗試捲動查看效果。

捲軸軌道使用表面色彩。

捲軸拇指使用分隔線色彩。

懸停時會有顏色變化效果。

整體設計簡潔現代。

符合 Material Design 規範。

這是最後一行內容。

捲軸樣式代碼

/* 自訂捲軸 */
::-webkit-scrollbar {
  width: 8px;
  height: 8px;
}

::-webkit-scrollbar-track {
  background: var(--surface);
}

::-webkit-scrollbar-thumb {
  background: var(--divider);
  border-radius: 4px;
}

::-webkit-scrollbar-thumb:hover {
  background: var(--outline);
}

📦 容器與無障礙性

響應式容器和螢幕閱讀器支援

.container 容器示範

這是一個使用 .container 類別的容器，會自動居中並具有最大寬度限制。

在不同螢幕尺寸下會自動調整寬度。

無障礙性支援

包含螢幕閱讀器專用的 .sr-only 類別：

這段文字只有螢幕閱讀器可以讀取 這段文字所有人都可以看到

容器與無障礙性代碼

/* 響應式容器 */
.container {
  width: 100%;
  max-width: var(--container-width);
  margin: 0 auto;
  padding: 0 var(--page-padding);
}

/* 螢幕閱讀器專用 */
.sr-only {
  position: absolute;
  width: 1px;
  height: 1px;
  padding: 0;
  margin: -1px;
  overflow: hidden;
  clip: rect(0, 0, 0, 0);
  white-space: nowrap;
  border: 0;
}
