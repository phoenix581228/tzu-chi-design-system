📖 Components 使用指南

🎯 組件設計原則

  • 所有組件都基於CI色彩系統設計
  • 保持一致的間距和圓角規範
  • 支援無障礙性和鍵盤導航
  • 響應式設計適配各種螢幕尺寸

🔧 組件修飾符

  • 尺寸：.btn-sm, .btn-lg (小、大尺寸)
  • 狀態：.active, .disabled, .loading
  • 變體：.outline, .ghost, .solid
  • 色彩：.primary, .secondary, .success, .error

🎨 自訂主題

組件使用CSS變數，可輕鬆自訂主題：

/* 自訂按鈕主色 */
.btn-custom {
  --btn-bg: var(--your-custom-color);
  --btn-text: var(--your-text-color);
  --btn-hover: var(--your-hover-color);
}

📱 響應式建議

  • 移動端使用較大的觸控目標 (44px+)
  • 調整按鈕和表單元素的間距
  • 簡化複雜的導航結構
  • 確保文字在小螢幕上的可讀性
