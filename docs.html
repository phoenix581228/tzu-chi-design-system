🚀 快速開始

慈濟設計系統 v2.0.0 是基於 Material Design 3.0 的企業級 CSS 框架， 提供完整的色彩系統、組件庫和工具類。

✨ 核心特色

  • 完整的 Material Design 3.0 實現
  • 模組化 CSS 架構（7個獨立模組）
  • 2000+ 預定義工具類
  • 完整的響應式設計支援
  • 深色/淺色主題切換
  • 無障礙設計支援（WCAG 2.1 AA）

📦 系統需求

💡 兼容性

支援所有現代瀏覽器，包括 Chrome 90+、Firefox 88+、Safari 14+、Edge 90+

📥 安裝指南

1
CDN 引入（推薦）

最簡單的使用方式，直接在 HTML 中引入完整的設計系統

<!-- 完整版本 --> <link rel="stylesheet" href="https://phoenix581228.github.io/tzu-chi-design-system/themes/material-design.css"> <!-- 或使用 v2.0.0 版本 --> <link rel="stylesheet" href="https://phoenix581228.github.io/tzu-chi-design-system/tzu-chi-design-system-v2.0.0.css">
2
模組化引入

按需引入特定模組，減少檔案大小

<!-- 基礎模組 --> <link rel="stylesheet" href="themes/variables.css"> <link rel="stylesheet" href="themes/base.css"> <!-- 功能模組（按需引入） --> <link rel="stylesheet" href="themes/colors.css"> <link rel="stylesheet" href="themes/typography.css"> <link rel="stylesheet" href="themes/layout.css"> <link rel="stylesheet" href="themes/components.css">
3
下載使用

下載完整專案到本地使用

# 使用 Git 複製 git clone https://github.com/phoenix581228/tzu-chi-design-system.git # 或直接下載 ZIP 檔案 # https://github.com/phoenix581228/tzu-chi-design-system/archive/main.zip

🎯 使用方法

基礎 HTML 結構

<!DOCTYPE html> <html lang="zh-TW"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>我的網站</title> <link rel="stylesheet" href="themes/material-design.css"> </head> <body class="material-design-system"> <!-- 您的內容 --> </body> </html>
⚠️ 重要提醒

請確保在 <body> 標籤上添加 material-design-system 類別

常用組件範例

<button class="btn btn-primary">主要按鈕</button> <button class="btn btn-secondary">次要按鈕</button> <button class="btn btn-outline">輪廓按鈕</button>

🎨 自訂主題

覆蓋 CSS 變數

:root { /* 覆蓋主要色彩 */ --primary-500: #your-brand-color; --secondary-500: #your-secondary-color; /* 覆蓋字體 */ --font-primary: 'Your Custom Font', sans-serif; /* 覆蓋邊角半徑 */ --border-radius: 12px; }

深色主題切換

// JavaScript 主題切換 function toggleTheme() { document.body.classList.toggle('light-theme'); // 保存用戶偏好 const theme = document.body.classList.contains('light-theme') ? 'light' : 'dark'; localStorage.setItem('theme', theme); } // 載入保存的主題 document.addEventListener('DOMContentLoaded', function() { const savedTheme = localStorage.getItem('theme'); if (savedTheme === 'light') { document.body.classList.add('light-theme'); } });

💡 最佳實踐

性能優化

  • 使用模組化引入，只載入需要的 CSS 模組
  • 在生產環境中使用壓縮版本
  • 利用瀏覽器快取機制

無障礙設計

  • 確保足夠的色彩對比度（遵循 WCAG 2.1 AA）
  • 為互動元素提供焦點指示器
  • 使用語義化的 HTML 標籤
  • 提供鍵盤導航支援

響應式設計

/* 使用響應式工具類 */ <div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3"> <div class="p-4">內容 1</div> <div class="p-4">內容 2</div> <div class="p-4">內容 3</div> </div>
